## Description

API REST para el **Sistema de Reservas de Salas de Reuniones** construido con [NestJS](https://github.com/nestjs/nest).

### Características
- ✅ **Autenticación JWT** con registro y login
- ✅ **Base de datos PostgreSQL** con TypeORM
- ✅ **Entidades**: Users, Rooms, Reservations
- ✅ **Validaciones** con class-validator
- ✅ **CORS habilitado** para desarrollo
- ✅ **Manejo de errores específico**

## Instalación y Configuración

### 1. Instalar dependencias
```bash
npm install
```

### 2. Instalar librerías adicionales
```bash
# PostgreSQL driver
npm install pg @types/pg

# bcryptjs para hash de contraseñas (ya incluido)
npm install bcryptjs @types/bcryptjs
```

### 3. Configuración de PostgreSQL
1. **Instalar PostgreSQL** en tu sistema
2. **Crear base de datos**: `booking_db`
3. **Configurar credenciales** en el archivo `.env`:
   ```env
   DATABASE_URL=postgresql://username:password@localhost:5432/booking_db
   ```

### 4. Variables de entorno (opcional)
Crea un archivo `.env` para personalizar la configuración:
```env
PORT=3000
DATABASE_URL=sqlite:///./database.sqlite
JWT_SECRET=tu-secret-super-seguro-aqui
JWT_EXPIRES_IN=24h
```

## Ejecutar la aplicación

```bash
# Modo desarrollo (recomendado)
npm run start:dev

# Modo producción
npm run start:prod

# Compilar sin ejecutar
npm run build
```

El servidor se ejecutará en: `http://localhost:3000`

## Run tests
